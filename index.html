CR16 Processor - Duck Hunt Game

Group project consisting of a 16-bit processor (Compact RISC ISA). Designed and built from scratch on an Intel Cyclone V FPGA board using Verilog. We built the Assembler and GlyphMaker program (Java) to combile the Duck Hunt application assembly instructions to run on the processor. The user I/O is a Nintendo Zapper gun for our custom Duck Hunt Game.

CE group project of a LIDAR 3D scanning robotic system. Designed and built from scratch with the robot set on top of a tripod to generate point cloud data.
